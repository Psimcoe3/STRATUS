The first step in writing a VB.NET program with Visual Studio 2019 is to choose a project type and create a new project.

1.  From the File menu, select New![](https://help.autodesk.com/cloudhelp/2023/ENU/Revit-API/images/ac.menuaro.gif)Projectâ€¦.
2.  In the Installed Templates frame, under Other Languages, click Visual Basic.
3.  In the right-hand frame, select Class Library. This example assumes that your project location is: C:\\Samples.
4.  In the Name field, type HelloWorld as the project name.
5.  Ensure that the .NET Framework 4.8 is selected at the top of the dialog.
6.  Click OK. ![](https://help.autodesk.com/cloudhelp/2023/ENU/Revit-API/images/GUID-A0AE601B-37D7-47BD-AFE8-5EE62B851958-low.png)**Figure 172: New Project dialog box**
