# Remarks

A new created ColorFillSchemeEntry should be assigned proper value before it is applied to a  [!:Autodesk::Revit::DB::ColorFillScheme]  . If the  [Color](51bf35c9-3ae3-7fef-2bfb-0b9007742126.htm)  and  [Caption](b0d11e7a-c253-a4a9-3d20-ec47577c78dd.htm)  properties are not set, Revit will generate them automatically after the ColorFillSchemeEntry is applied to the ColorFillScheme.
