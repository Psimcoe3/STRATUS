#### Parameters

height
:   Type:  [System Double](http://msdn2.microsoft.com/en-us/library/643eft0t)

pitch
:   Type:  [System Double](http://msdn2.microsoft.com/en-us/library/643eft0t)    
     The pitch, or vertical distance traveled in one rotation.

baseFinishingTurns
:   Type:  [System Int32](http://msdn2.microsoft.com/en-us/library/td2s409d)

:   Type:  [System Int32](http://msdn2.microsoft.com/en-us/library/td2s409d)
