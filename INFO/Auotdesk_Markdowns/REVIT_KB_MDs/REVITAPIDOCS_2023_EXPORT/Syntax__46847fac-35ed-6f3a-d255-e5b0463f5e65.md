# Remarks

After this method has been invoked, call:

* [GetConvertedFabricationParts](3461800e-90b9-2961-313d-ceb5c6e48b69.htm)  to get a set of element identifiers for the newly created fabrication parts.
* [GetElementsWithOpenConnector](1b8323a9-dd24-c818-e74c-e29b346000d3.htm)  to get a set of fabrication part or MEP design element identifiers with open connectors, caused by fittings failing to convert.
