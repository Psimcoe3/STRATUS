# Remarks

Title is derived from the document's filename, not including the filename extension. Note that returned title will be defaulted if no document saved. Note that returned string may be empty if a document is detached. See  [IsDetached](0792283e-f112-0a57-d0d9-e79e6b9ea5b9.htm)  .
