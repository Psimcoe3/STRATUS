This method is used to get a Reference from an adjacent MassZone that is coincident to a Reference of this MassZone.

#### Parameters

referenceToZoneFace
:   Type:  [Autodesk.Revit.DB Reference](d28155ae-817b-1f31-9c3f-c9c6a28acc0d.htm)    
     Reference to face of MassZone.

#### Return Value

Reference to coincident face of adjacent MassZone or    a null reference (  Nothing  in Visual Basic)  if there is no adjacent MassZone face Reference.

# Remarks

The result is always a Reference to a Face. The Reference Type should report as REFERENCE\_TYPE\_SURFACE. Currently Revit improperly reports it as REFERENCE\_TYPE\_NONE. The results are always references to Faces.
