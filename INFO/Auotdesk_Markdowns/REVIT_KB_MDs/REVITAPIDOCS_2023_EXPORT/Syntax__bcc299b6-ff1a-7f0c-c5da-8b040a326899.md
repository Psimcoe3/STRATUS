# Remarks

The parametric equation of the cone is S(u, v) = center + v\*[sin(halfAngle)(cos(u)\*xVec + sin(u)\*yVec) + cos(halfAngle)\*zVec]. Only the branch of the cone with v >= 0 should be used.
