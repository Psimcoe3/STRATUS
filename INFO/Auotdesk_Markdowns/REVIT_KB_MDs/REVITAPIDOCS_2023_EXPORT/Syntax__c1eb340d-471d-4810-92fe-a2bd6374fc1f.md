#### Return Value

# Remarks

The returned collection does not include dimensions labeled by this global parameter. To get the labeled dimensions use the  [GetLabeledDimensions](97d29291-74c4-2da5-2ac5-2fa0c0ac9d0c.htm)  method. Other global parameters that may be affected by this one (in their formulas) are also excluded from the returned collection. To get the set of affected global parameters use the  [GetAffectedGlobalParameters](2028f8a1-2691-e921-8a56-882b1e4080f3.htm)  method.
