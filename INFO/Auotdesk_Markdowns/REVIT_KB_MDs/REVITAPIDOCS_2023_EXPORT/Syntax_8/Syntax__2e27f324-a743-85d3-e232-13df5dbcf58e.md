# Remarks

Apply the view template parameters is a one-time operation, the association between the views is not preserved. To get all template parameters call  [GetTemplateParameterIds](64761c8c-ed01-65b6-2b05-ebd7b02acd77.htm)  on the template. To get parameters that are not controlled by the template call  [GetNonControlledTemplateParameterIds](a34bb9cf-9a1d-a3e7-b04e-78bca30ecf4e.htm)  on the template. To get current view template of this view use property  [ViewTemplateId](2559f20b-87d4-e879-3139-7f555b251b71.htm)  .
