#### Return Value

# Remarks

Returns -1 if there is a mix of list start values in the range or if there are paragraphs in different lists or sub-lists in the range or if there are paragraphs of type  [None](7163554f-3446-22eb-afa4-5490d5df29c8.htm)  or  [Bullet](7163554f-3446-22eb-afa4-5490d5df29c8.htm)  in the range.
