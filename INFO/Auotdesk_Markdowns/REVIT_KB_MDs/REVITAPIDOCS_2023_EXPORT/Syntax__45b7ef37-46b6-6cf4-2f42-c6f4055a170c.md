# Remarks

Each effect instance should be used with geometry of a matching vertex format. Only a subset of effect instance parameters is relevant to geometry of a particular vertex format. If the vertex format does not specify vertex color (  [VertexPosition](718e49aa-9e17-6f2d-2013-141b5cfeefdd.htm)  or  [VertexPositionNormal](a40efda7-6e2f-a455-f65e-02b10b0bc1b4.htm)  ) the following parameters should be set:

* [Color](6f5f2e39-a850-5a55-ee09-7d2856209d84.htm)
* [Transparency](6866ee5e-23ea-9b05-a6f4-74ba3795b81d.htm)

If the vertex format specifies vertex normals (  [VertexPositionNormal](a40efda7-6e2f-a455-f65e-02b10b0bc1b4.htm)  or  [VertexPositionNormalColored](aa354e03-2b25-b5a4-5634-c3518518c0d3.htm)  ) the following parameters should be set:

* [Ambient Color](8b664a39-8174-b5c3-f4b7-67f71ffaaf6b.htm)
* [Diffuse Color](7302a193-6396-93a7-a36e-0bde4f2a119c.htm)
* [Specular Color](715cb93c-e62a-280d-0095-47950d1cce7f.htm)
* [Glossiness (specular exponent)](8b664a39-8174-b5c3-f4b7-67f71ffaaf6b.htm)
* [EmissiveColor](00b45e82-3bd5-b592-66e9-36364628c59b.htm)
* [Transparency](6866ee5e-23ea-9b05-a6f4-74ba3795b81d.htm)

If the vertex format specifies a color, then the geometry will be colored according to the color of each vertex. Otherwise, the geometry's color will come from the effect instance parameters. Similarly, the transparency parameter affects all of the geometry associated with the effect, while the transparency of the individual vertices can also be controlled via the transparency component of their colors.
