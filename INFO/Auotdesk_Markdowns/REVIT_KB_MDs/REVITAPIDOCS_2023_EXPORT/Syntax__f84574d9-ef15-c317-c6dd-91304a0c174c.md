# Remarks

This method is only called for photo-rendering export (a custom exporter that implements  [IPhotoRenderContext](d09d4ea2-1090-f2b9-8073-5fb8a796babf.htm)  ). When an RPC object is encountered for a model context export (a custom exporter that implements  [IModelExportContext](4309af43-f04e-4e42-2539-3fd1d64cdc6d.htm)  ), the RPC object will be provided as a polymesh (via  [OnPolymesh(PolymeshTopology)](6a060c37-3225-217e-b150-2eaea3a22c29.htm)  ).
