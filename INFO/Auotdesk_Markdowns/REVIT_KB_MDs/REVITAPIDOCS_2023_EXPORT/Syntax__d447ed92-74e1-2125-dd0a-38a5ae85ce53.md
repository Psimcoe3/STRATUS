# Remarks

If the family is not loaded, or if the family is loaded but unchanged, the situation will never trigger and  [OnFamilyFound(Boolean, Boolean )](515baa3b-3a13-bb28-6c94-a84651b4dbfb.htm)  and  [OnSharedFamilyFound(Family, Boolean, FamilySource , Boolean )](7bb3a582-a7c0-1f3d-ebe3-33a1bfa443fc.htm)  will not be called. Only if the family is loaded and changed should the interface methods be called.
