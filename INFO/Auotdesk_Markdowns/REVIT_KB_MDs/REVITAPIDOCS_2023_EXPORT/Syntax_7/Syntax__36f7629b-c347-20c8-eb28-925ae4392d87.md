# Remarks

This function returns the list type of all paragraphs contained in the given range. Returns  [None](7163554f-3446-22eb-afa4-5490d5df29c8.htm)  if the paragraph is not in a list. Returns  [Mixed](7163554f-3446-22eb-afa4-5490d5df29c8.htm)  if the list types don't match between the paragraphs.
