#### Return Value

True if the sub-transaction has already been committed or rolled back, False otherwise.

# Remarks

A sub-transaction is ended by completing either the  [Commit](65a0359a-ef13-e7aa-7d5c-7470fe177848.htm)  or  [RollBack](3de65ee0-50f1-c601-62f9-c77479b08418.htm)  method. Another way of testing whether a sub-transaction has ended is by testing its current status returned from (see  [GetStatus](3da7e561-66d4-97d5-ed12-7ff77b478421.htm)  ). It must be either 'TransactionStatus.Committed' or 'TransactionStatus.RolledBack'
