# Remarks

This change is permanent until the elements are unhidden. All elements in the set must be currently unhidden and must be allowed to be hidden. An application can check this with  [IsHidden(View)](2c3d4123-fded-cd5f-ed0d-12b1e1a3ce42.htm)  and  [CanBeHidden(View)](887010c4-de58-96b6-0931-4c226e6b142b.htm)  .

Some elements cannot be hidden directly. Examples include: group, array, constraint, edit cut profile, face splitter and link to external documents.

To hide a group completely, you must also include all members of all groups and nested groups in your input.
