# Remarks

The value assigned to the TextBox can be a  [String](http://msdn2.microsoft.com/en-us/library/s1wwdcbf)  or any other data type. If the type is not a String the TextBox will display the return of the ToString() method. When the text is edited in the UI, the type of Value will always be  [String](http://msdn2.microsoft.com/en-us/library/s1wwdcbf)  .
