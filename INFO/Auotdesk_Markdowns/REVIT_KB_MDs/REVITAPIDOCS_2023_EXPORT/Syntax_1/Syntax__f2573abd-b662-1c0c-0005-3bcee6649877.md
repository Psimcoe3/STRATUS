#### Return Value

The short display name of the external resource.

# Remarks

For an external resource, such as a Rvt Link loaded from an external server, which has a full display path such as "My Server://Nested/Nested\_1.rvt", this function returns "Nested\_1.rvt". For an external resource, such as a Rvt Link loaded from the "built-in" server, which has a full display path such as "c:\LocalLinks\Link\_1.rvt", this function returns "Link\_1.rvt".
