# Remarks

When getting this property, it returns a copy of the  [ExportFontInfo](c3dc100c-0e4d-419d-5cbd-1d59f149490c.htm)  from the table. In order to make changes to the  [ExportFontInfo](c3dc100c-0e4d-419d-5cbd-1d59f149490c.htm)  and use those settings during export, set the modified  [ExportFontInfo](c3dc100c-0e4d-419d-5cbd-1d59f149490c.htm)  back into the table using the same key.
