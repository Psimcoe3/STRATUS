#### Parameters

iSegment
:   Type:  System Int32    
     Index of the segment (0 to NumberOfSegments - 1).

paramId
:   Type:  [Autodesk.Revit.DB ElementId](44f3f7b1-3229-3404-93c9-dc5e70337dd6.htm)
