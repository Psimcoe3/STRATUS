#### Field Value

The id of the template view that controls this view's parameters. InvalidElementId if there is no template set for this view.

# Remarks

Parameters controlled by the template cannot be changed directly in this view. Any change to controlled parameters in the template will cause change of the corresponding parameters in this view. Use  [GetTemplateParameterIds](64761c8c-ed01-65b6-2b05-ebd7b02acd77.htm)  on the template view to get a list of all template parameters. Use  [GetNonControlledTemplateParameterIds](a34bb9cf-9a1d-a3e7-b04e-78bca30ecf4e.htm)  on the template view to get parameters that are not controlled by the template.
