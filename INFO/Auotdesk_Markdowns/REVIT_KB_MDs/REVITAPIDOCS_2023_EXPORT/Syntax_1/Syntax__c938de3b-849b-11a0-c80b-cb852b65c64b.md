# Remarks

If  [LockProportions](2eda8038-9240-00f5-ffa3-a65d8b4df5f2.htm)  is true, then changes to the  WidthScale  will also result in changes to the  [HeightScale](c0c41a38-7554-4472-479a-794b561a6534.htm)  .
