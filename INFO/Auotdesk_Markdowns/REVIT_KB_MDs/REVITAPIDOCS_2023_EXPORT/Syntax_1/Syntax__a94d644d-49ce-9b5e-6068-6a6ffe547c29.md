#### Parameters

handle
:   Type:  [Autodesk.Revit.DB.Structure RebarConstrainedHandle](08b4c4a3-3bb9-0801-9cc8-cd5420a306d9.htm)

:   Type:  System.Collections.Generic IList   [Reference](d28155ae-817b-1f31-9c3f-c9c6a28acc0d.htm)
