#### Return Value

True if the view has a view type that allows Reveal Constraints mode to be activated.

# Remarks

Reveal Constraints mode can only be activated for the following viewtypes:

* 3D Views
* Area Plans
* Ceiling Plans
* Detail Views
* Drafting Views
* Elevations
* Floor Plans
* Sections
* Structural Plans
