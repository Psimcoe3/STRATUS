#### Return Value

The section box. Note that the section box can be rotated and transformed and thus you will need to use  [Transform](297887ab-69bb-548e-cfb6-a3a23f410604.htm)  to interpret the coordinates of the corners or sides of the box in model coordinates.

# Remarks

The section box cuts the model in this view by its boundaries.
