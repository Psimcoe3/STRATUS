Returns ElementIds for Category elements which are excluded (not taken into account) by route calculation. These categories are always excluded, regardless of the  [EnableIgnoredCategoryIds](c146653a-ed81-ee52-7102-a6c2ad8a4d7c.htm)  value. If an excluded category has sub-categories, then the sub-categories will be excluded as well.
