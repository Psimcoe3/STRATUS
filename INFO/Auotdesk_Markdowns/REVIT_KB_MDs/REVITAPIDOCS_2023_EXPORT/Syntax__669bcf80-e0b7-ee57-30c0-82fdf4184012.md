# Remarks

Like AddParameter(), this function introduces a parameter into the shape definition, but the parameter's value is driven by a formula. Formula parameters cannot be used in constraints to drive the shape. The formula is in the same format as in Revit families. The formula is allowed to refer to other parameters that are already in the definition, plus the builtin parameters REBAR\_INSTANCE\_BAR\_DIAMETER (bar nominal diameter), REBAR\_INSTANCE\_BAR\_MODEL\_DIAMETER (bar model diameter), REBAR\_INSTANCE\_BEND\_DIAMETER, REBAR\_SHAPE\_START\_HOOK\_LENGTH, REBAR\_SHAPE\_START\_HOOK\_OFFSET, REBAR\_SHAPE\_PARAM\_START\_HOOK\_TAN\_LEN, REBAR\_SHAPE\_PARAM\_END\_HOOK\_TAN\_LEN, REBAR\_SHAPE\_END\_HOOK\_LENGTH, REBAR\_SHAPE\_END\_HOOK\_OFFSET, REBAR\_ELEM\_LENGTH, REBAR\_ELEM\_TOTAL\_LENGTH, and REBAR\_ELEM\_QUANTITY\_OF\_BARS.
