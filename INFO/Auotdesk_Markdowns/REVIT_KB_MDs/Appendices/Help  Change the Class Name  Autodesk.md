To change the class name, complete the following steps:

1.  In the Solution Explorer, right-click Class1.vb to display a context menu.
2.  From the context menu, select Rename. Rename the file HelloWorld.vb.![](https://help.autodesk.com/cloudhelp/2023/ENU/Revit-API/images/GUID-5F89A48C-D52D-4F84-B4FB-8A363CAC52C7-low.png)
3.  The following message will appear. Click Yes.![](https://help.autodesk.com/cloudhelp/2023/ENU/Revit-API/images/RenameClass1.png)
4.  In the Solution Explorer, double-click HelloWorld.vb to open it for editing.**Figure 174: Change the class name**
